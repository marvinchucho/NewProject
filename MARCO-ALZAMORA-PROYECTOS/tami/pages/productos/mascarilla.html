INFORMACIÓN DEL PRODUCTO

Confort


El respirador está diseñado para una protección cómoda contra el polvo, el polen y otras fuentes de contaminación. El turbo ventilador integrado tiene un cambio de volumen de aire ajustable de 2 velocidades que permite un flujo de aire suave y trae un aire fresco positivo, por lo que no se siente transpirable incluso después de un largo tiempo de uso. La batería de litio integrada puede funcionar 4-6 horas por carga.


Diseño Ergonómico


El dispositivo anti polvo anti alérgico sigue de cerca el principio de la ergonomía, 3D se adapta a la cara y mantiene un buen sellado, por lo que no empañará las gafas mientras usa este filtro de aire. La cubierta del filtro de aire de succión magnética es fácil de desmontar, lo que le permite reemplazar los filtros fácilmente. El filtro HEPA de estructura de 4 capas (Carbono, HEPA, papel esterilizado, tela no tejida) puede bloquear eficazmente el 98% del contenido de contaminación del aire.


Uso Universal


El ambientador anticontaminación es adecuado para proyectos de mejora del hogar: Como ambientador de pintura, ambientador para limpieza, carpintería, siega, etc. También se puede utilizar como dispositivo antialergénico para filtrar el polen y como accesorio de aire para la gripe. Nuestros clientes también utilizan el ambientador facial para correr, ciclismo y otras actividades al aire libre. El ambientador facial antipolvo es unisex, con 2 auriculares de silicona estirados y 2 clips para ajustar la mejor longitud según las diferentes caras.


Reutilizable y reemplazable


El filtro de aire de seguridad es reutilizable, la silicona del sello facial es extraíble y lavable, el filtro de aire es reemplazable, para ofrecer un aire más limpio para proteger la boca, la nariz y los pulmones. Es cómodo para llevarlo durante mucho tiempo y es fácil de limpiar


Consejos


Si lo usas 1-3 horas al día, se recomienda reemplazar el filtro una vez al mes.


Si lo usa durante menos de 1 hora al día, se recomienda reemplazar el filtro cada 45 días.


Contenido del paquete


1 x filtro de aire


2 x filtros Chips


2 x Clips


1 x Cable USB


1 x Manual del usuario
